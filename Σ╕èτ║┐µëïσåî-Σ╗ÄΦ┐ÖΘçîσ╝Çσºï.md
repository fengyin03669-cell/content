# 🚀 上线手册：从现在到"朋友能用"

读完这份,你就能把这个工具变成一个公开网址,发给任何人用,
**他们不需要你的电脑、不需要你的 Claude 账号、不需要懂技术**。

全程在网页上点点点,不用敲任何命令。预计 30–40 分钟。

---

## 你需要准备的三个免费账号

| 账号 | 用来做什么 | 在哪注册 |
|------|-----------|---------|
| 通义千问（阿里云百炼） | 提供 AI 能力（写文案） | bailian.console.aliyun.com |
| GitHub | 存放代码 | github.com |
| Render | 把代码变成公开网址 | render.com |

三个都免费。下面一步步来。

═══════════════════════════════════════════════

## 第 1 步：拿到 AI 钥匙（通义千问 API Key）

1. 浏览器打开 https://bailian.console.aliyun.com
2. 用手机号注册 / 登录阿里云账号
3. 按提示"开通百炼"（免费开通）
4. 左侧菜单找到 **API-KEY**（或"我的 API-KEY"）
5. 点 **创建新的 API-KEY**，会得到一串 `sk-` 开头的字符
6. **复制保存到记事本**——这串就是你的钥匙，等下要用

> 💡 新账号有大量免费额度，前期测试基本不花钱。
> ⚠️ 这串钥匙别发给别人、别截图发群里，它能花你的钱。

═══════════════════════════════════════════════

## 第 2 步：把代码放上 GitHub

1. 打开 https://github.com 注册 / 登录
2. 解压你下载的 `huaguang-app.zip`，得到一个 huaguang-app 文件夹
3. 在 GitHub 右上角点 **+** → **New repository（新建仓库）**
4. 仓库名随便填，比如 `huaguang-app`，下面选 **Public**，点 **Create repository**
5. 进入空仓库页面，点 **uploading an existing file**（上传已有文件的链接）
6. 把解压后 huaguang-app 文件夹里的**所有内容**（server.js、package.json、
   public 文件夹、README.md 等）**全选拖进网页**
   - ⚠️ 注意：拖的是文件夹"里面"的东西，不是文件夹本身
   - ⚠️ 不要上传 node_modules 文件夹（如果有的话，跳过它）
7. 拖完后点底部绿色按钮 **Commit changes**

代码就上传好了。

═══════════════════════════════════════════════

## 第 3 步：在 Render 上线，得到公开网址

1. 打开 https://render.com，点 **Get Started** / 用 **GitHub 账号登录**
   （这样它能直接读到你刚上传的代码）
2. 登录后，点右上角 **New +** → 选 **Web Service**
3. 选 **Build and deploy from a Git repository**，点 Next
4. 找到你刚建的 `huaguang-app` 仓库，点 **Connect**
5. 接下来填几个设置（大部分会自动填好，核对一下）：
   - **Name**：随便起，会成为网址的一部分，比如 `huaguang`
   - **Build Command（构建命令）**：填 `npm install`
   - **Start Command（启动命令）**：填 `npm start`
   - **Instance Type（套餐）**：选 **Free（免费）**
6. 往下滚动找到 **Environment Variables（环境变量）**，点 **Add Environment Variable**，
   添加这两条：

   | Key（名称） | Value（值） |
   |------------|------------|
   | `PROVIDER` | `qwen` |
   | `QWEN_API_KEY` | 粘贴你第 1 步保存的 `sk-` 钥匙 |

7. 点最底部 **Create Web Service**
8. 等几分钟（屏幕会滚动显示进度），出现 **Live** 字样就成功了
9. 页面顶部会有一个网址，长得像 `https://huaguang.onrender.com`
   —— **这就是你的公开网址！**

═══════════════════════════════════════════════

## 第 4 步：发给朋友

把那个 `.onrender.com` 网址复制，微信 / 邮件发给朋友。
他们用手机或电脑浏览器打开就能用,**不需要登录任何账号**。

手机上还能"添加到主屏幕",像 App 一样用。

═══════════════════════════════════════════════

## ⚠️ 免费版的一个小脾气

Render 免费套餐有个特点：**超过 15 分钟没人用，它会"休眠"**。
下次有人打开时，第一次要等 30–50 秒"唤醒"，之后就正常了。

- 如果你只是给几个朋友偶尔用 → 免费版完全够，忍一下首次的几十秒即可
- 如果要很多人随时流畅用 → 可在 Render 升级到付费套餐（几美元/月），不再休眠

═══════════════════════════════════════════════

## 🔁 以后想换模型 / 改钥匙

不用重新部署。去 Render 你的服务页面 → 左侧 **Environment** →
改 `PROVIDER` 的值（qwen / kimi / doubao / deepseek / anthropic）
并填好对应的钥匙（变量名见 .env.example）→ 保存，它会自动重新上线。

## 🆘 卡住了怎么办

任何一步卡住，把你看到的**页面截图或报错文字**发给我，我帮你看是哪里的问题。
最常见的两个问题：
- 部署后打开是报错页 → 多半是环境变量 `QWEN_API_KEY` 没填对，回第 3 步第 6 点检查
- 生成时提示"AI 接口调用失败" → 多半是钥匙余额 / 额度问题，去阿里云百炼后台看一下

你已经走完最难的部分了（代码全做好了）。剩下就是照着上面点几下，加油 💪
